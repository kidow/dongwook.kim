# dongwook.kim

Personal portfolio site built with Next.js App Router.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui

## Development

```bash
pnpm dev
```

## Quality Checks

```bash
pnpm lint
pnpm type-check
pnpm test
pnpm test:e2e
pnpm build
```

## Key Directories

- `app`: routes and API handlers
- `components`: UI and feature components
- `components/ui`: shadcn/ui primitives
- `components/Widget`: homepage widgets
- `components/Editor`: memo editor
- `utils`: shared utilities and API wrappers
- `docs`: migration and integration docs
