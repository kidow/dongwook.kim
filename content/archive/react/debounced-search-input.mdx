## Problem

검색 입력창에서 key stroke마다 네트워크 요청을 보내면 불필요한 렌더링/요청이 증가할 수 있습니다.

## Snippet

```tsx
const [query, setQuery] = useState('')
const [debouncedQuery] = useDebounce(query, 300)

useEffect(() => {
  fetchPosts(debouncedQuery)
}, [debouncedQuery])
```

## Notes

- 300ms 지연은 기본값이며, 화면 특성에 따라 조정합니다.
- 로딩 상태는 `query`가 아닌 `debouncedQuery` 기준으로 관리합니다.
